# Supervised Learning App

A Python desktop application that automates supervised machine learning tasks through a simple graphical user interface (GUI). Users can upload training and testing datasets, select either **Classification** or **Regression**, and let the application automatically choose the best-performing machine learning model.

## Features

* Simple Tkinter-based GUI
* Upload `data.csv` (training dataset)
* Upload `test.csv` (testing dataset)
* Select Classification or Regression
* Automatic data preprocessing
* Trains and compares multiple machine learning algorithms
* Automatically selects the best-performing model
* Generates predictions for the test dataset
* Exports prediction results as `output/output.csv`
* Generates a detailed `output/report.txt` with evaluation metrics

## Machine Learning Algorithms

### Classification

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Gaussian Naive Bayes

### Regression

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor
* K-Nearest Neighbors Regressor
* Support Vector Regressor (SVR)

## Installation

1. Clone the repository.

```bash
git clone https://github.com/ManasviP10/SupervisedLearningApp.git
```

2. Navigate to the project folder.

```bash
cd SupervisedLearningApp
```

4. Run the application.

```bash
python main.py
```

## How to Use

1. Launch the application.
2. Select the training dataset (`data.csv`).
3. Select the testing dataset (`test.csv`).
4. Choose either **Classification** or **Regression**.
5. Click the **START** button.
6. Wait for the training process to complete.
7. View the generated files:

   * `output/output.csv`
   * `output/report.txt`

## Output Files

### output.csv

Contains the predictions generated by the best-performing machine learning model.

### report.txt

Includes:

* Selected algorithm
* Performance metrics
* Accuracy or R² score
* Precision, Recall, and F1-score (classification)
* MAE, MSE, RMSE, and R² (regression)
* Execution time
* Comparison of all evaluated models
